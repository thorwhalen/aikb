class aikb.base.KnowledgeFiles(provider: KnowledgeBaseProvider, *, project_id: str)[source]

Dict-like access to knowledge files in an AI project.

Keys are filenames (str), values are file contents (str).

>>> import tempfile, os
>>> store = LocalFiles(tempfile.mkdtemp())
>>> store['notes.md'] = '# Notes'
>>> store['notes.md']
'# Notes'
>>> list(store)
['notes.md']
>>> len(store)
1
>>> del store['notes.md']
>>> list(store)
[]
class aikb.base.KnowledgeMall(stores: dict | None = None, /, **named_stores)[source]

A read-only mapping of names to KnowledgeFiles stores.

Useful for multi-project or multi-platform workflows.

>>> import tempfile
>>> mall = KnowledgeMall(staging=LocalFiles(tempfile.mkdtemp()))
>>> list(mall)
['staging']
>>> isinstance(mall['staging'], KnowledgeFiles)
True
